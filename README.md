# Flujo_de_datos_de_SQL_Grupo1
Proyecto de limpieza de datos utilizando la base de datos Sakila. La extracción y preprocesamiento se realizan en SQL, seguidos de una limpieza y análisis en Python (Google Colab). El flujo incluye joins entre múltiples tablas, validación de datos, normalización y generación de un dataset limpio para su posterior análisis.
